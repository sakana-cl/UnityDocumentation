Unity 基础

这部分是你开始使用Unity的关键。这将解释Unity界面,菜单项,使用资源,创造场景和发布构建。
This section is your key to getting started with Unity. It will explain the Unity interface, menu items, using assets, creating scenes, and publishing builds.

当你读完这一节中,您将了解Unity是如何工作的,如何有效地使用它,并一步步创建一个简单的游戏。
When you are finished reading this section, you will understand how Unity works, how to use it effectively, and the steps to put a basic game together.

 界面学习-Learning the Interface
有很多东西要学，所以要花一些时间学习界面，我们将每一个界面元素都会介绍。-There is a lot to learn, so take the time you need to observe and understand the interface. We will walk through each interface element together.

 资源工作流-Asset Workflow
这里我们将介绍Unity使用资源的步骤，这些步骤是综合的只作为基本动作的概述。Here we'll explain the steps to use a single asset with Unity. These steps are general and are meant only as an overview for basic actions.

 创建场景-Creating Scenes
场景包含你的游戏的对象。在每一个场景,你将使用游戏对象模块布置你的环境中,障碍,装饰,设计和构建你的场景。Scenes contain the objects of your game. In each Scene, you will place your environments, obstacles, and decorations, designing and building your game in pieces.
 编译发布-Publishing Builds
在任何时候当你创建游戏时，你可能想看看它效果怎么样，编译为独立版或网页版并运行。这部分讲解编译设置如何访问和为你的游戏创建不同的编译环境。At any time while you are creating your game, you might want to see how it looks when you build and run it outside of the editor as a standalone or web player. This section will explain how to access the Build Settings and how to create different builds of your games.

 教程-Tutorials
这些是Unity的在线教程，提供实用与真实项目建设的经验。These online tutorials will let you work with Unity while you follow along, providing hands-on experience with building real projects.
